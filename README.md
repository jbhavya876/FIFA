## Getting Started

### Prerequisites
- Node.js (20.19.3)
- npm 
- MetaMask, Phantom, or Coinbase Wallet (for testing crypto features)

### Installation
1. Clone the repository:
   ```bash
   cd epl-sports-betting-nolog
   npm install
   npm run dev
   ```
